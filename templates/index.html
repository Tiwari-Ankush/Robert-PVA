Robert - Personal Assistant

About Robert

Robert is a personal assistant designed to make your life easier. With Robert, you can perform various tasks such as playing music, fetching the latest news, checking the weather, and much more. Built with Python and powered by Flask, Robert is your go-to assistant for everyday tasks.

This project is a testament to the power of modern technology and creativity. Whether you're a developer or a tech enthusiast, Robert is here to assist you in your daily life.

Available Commands

  • Play [song name] - Play a specific song.
  • What's the weather? - Get the current weather.
  • Open [website] - Open a website (e.g., Open Google).
  • Take notes - Save and open a note.
  • Set volume [percentage] - Set system volume.
  • Set brightness [percentage] - Set screen brightness.
  • Breaking news - Fetch the latest news.
  • Shutdown/Restart/Lock/Sleep - Control your system.
